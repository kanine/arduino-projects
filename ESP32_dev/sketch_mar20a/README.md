# sketch_mar20a

Basic ESP32 hello-world sketch. Prints `"ESP32 is working!"` to the serial monitor once per second.

Used to verify a fresh board is alive and serial communication is working.

## Hardware

| Item | Details |
|---|---|
| Board | ESP32 WROOM (3.3V logic) |
| Extra hardware | None |

## OTA Firmware Updates

Not supported. Upload via USB only.

## Configuration

None. No `secrets.h` required.

## Serial Output

Baud rate: **115200**

```
ESP32 is working!
ESP32 is working!
...
```
