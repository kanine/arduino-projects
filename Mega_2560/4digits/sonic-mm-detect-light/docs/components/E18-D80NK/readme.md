# Infrared Obstacle Avoidance Sensor – E18-D80NK

Transmitter and receiver combined in a panel-mount photoelectric sensor. Adjustable detection distance via onboard potentiometer.

Specifications
- Model: E18-D80NK
- Input voltage: 5 V DC
- Current consumption: 25 mA (typical minimum) – 100 mA (maximum)
- Dimensions: 17 mm (diameter) × 45 mm (length)
- Cable length: 45 cm
- Detection: transparent or opaque objects (diffuse reflective)
- Sensing range: 3 cm – 80 cm (surface dependent; adjustable)
- Output type: NPN (normally high)
- Operating temperature: −25 °C to 55 °C

Pin Configuration:
- Brown Wire: +5V
- Blue Wire: GND
- Black Wire: DIGITAL OUTPUT – Open collector pull down (Use 10K pull up resistor to +5V)
